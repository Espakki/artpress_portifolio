02 Anatomia de cores
Surface Container
#201F1F
--surface-container
Fundo principal do favicon. Segunda camada do sistema de superfície — nunca usar preto puro.
On Surface
#E5E2E1
--on-surface
Fill do A geométrico. Off-white quente — nunca usar #FFFFFF puro, que quebra a atmosfera industrial.
Primary Container
#FF6B00
--primary-container
Ponto de assinatura da marca. Aparece em todas as 8 variações — é o elemento invariável do sistema.

07 Do's & Don'ts
Do — correto
Usar #201F1F como fundo — nunca preto puro #000000
Usar #E5E2E1 para o A — nunca branco puro #FFFFFF
Manter o ponto #FF6B00 em todas as variações
Exportar SVG com viewBox="0 0 64 64" como master
Ajustar rx proporcionalmente ao tamanho renderizado
Usar role="img" e aria-label="ArtPress" no SVG
Definir theme-color: #201F1F no manifest e meta tag
Don't — proibido
Não usar branco puro #FFFFFF como fundo do favicon
Não remover o ponto laranja — é a assinatura da marca
Não aplicar gradiente diretamente no SVG do favicon
Não usar border-radius: 50% (pill) — manter geométrico
Não redimensionar via CSS sem manter aspect ratio 1:1
Não usar o favicon sobre fundos laranja ou amarelos
Não alterar a geometria do A — proporções são fixas
08 Guia de uso nos produtos
Website / App Web
16px · 32px · SVG
Implementar via <link rel="icon"> no <head>. Priorizar SVG para suporte a Retina. O SVG escala perfeitamente sem pixelação. Definir theme-color #201F1F para colorir a barra do browser no mobile.
iOS / Apple
180px · apple-touch-icon
Usar apple-touch-icon.png em 180×180px. O iOS aplica automaticamente corners arredondados sobre o PNG — não pré-arredondar o arquivo. Fundo opaco #201F1F obrigatório (sem transparência).
Android / PWA
192px · 512px · maskable
Exportar versão maskable com safe zone de 40% (o A centralizado dentro do círculo seguro). No webmanifest, incluir ambos os propósitos: "purpose": "maskable any". Usar background_color #131313.
Email / Newsletter
32px · 64px · PNG inline
Clientes de email não suportam SVG. Usar PNG 64px com fundo opaco. Adicionar alt="ArtPress" obrigatório. Em assinaturas, preferir a versão 32px para manter proporção com o tamanho de fonte do rodapé.
Open Graph / Social
1200×630px · PNG
O favicon não é usado diretamente — mas o sistema de cores deve refletir na OG image. Usar #201F1F como fundo e o A geométrico em tamanho display como elemento central. Ponto laranja como elemento de assinatura.
Figma / Design Tools
64px · 512px · SVG component
Criar como componente Figma com propriedades: size (16/32/64/512), variant (dark/light/high-vis). Usar o SVG master como base. Definir os tokens de cor como estilos de cor do arquivo para reuso consistente.
