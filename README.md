# ArtPress Portfólio

Projeto inicial de portfólio TDE

## Estrutura

```text
portfolio/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── images/
    ├── profile.jpg
    └── projeto1.png
```

## Como usar

1. Abra o arquivo `portfolio/index.html` no navegador.
2. Edite estilos em `portfolio/css/style.css`.
3. Adicione scripts em `portfolio/js/script.js`.
4. Substitua as imagens em `portfolio/images/` pelas suas imagens reais.
